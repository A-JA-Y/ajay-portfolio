## 🏗️ Installation & Setup

### Prerequisites
- **Node.js** (v14 or higher)
- **npm** or **yarn** package manager

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/A-JA-Y/ajay-portfolio.git
   cd ajay-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
# Create optimized production build
npm run build

# Test production build locally
npm install -g serve
serve -s build
```
